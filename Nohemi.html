Ciberseguridad 01: Tipos de Malware

Tipo de Malware Función Impacto
Virus Auto replicable, infecta archivos Daño de datos al igual que de sistema
Gusanos Se propaga a través de las redes Interrupción de la red, agotamiento de recursos
Caballo de Troya Se disfraza de software legítimo Robo de datos, control de sistema
Software Espía Supervisa en secreto la actividad del usuario Violación de privacidad, robo de datos
Software Publicitario Muestra anuncios no deseados Molestia, degradación del rendimiento
Rootkit Ocultar software malicioso Acceso no autorizado, amenazas persistentes
Keylogger Registra las pulsaciones de teclas Robo de credenciales, filtración de datos
Fileless Malware
(Malware sin archivos)		 Corre desde la memoria Ataques sigilosos y difíciles de detectar
Cryptojacker Minería de criptomonedas Problemas de rendimiento, secuestro de recursos
Red de Bots (Botnet) Crea una red de dispositivos infectados Ataques DDoS, distribución de spam
Bomba Lógica Ejecuta código en el disparador Destrucción de datos, daños al sistema
Wiper (Limpiador) Destruye datos Pérdida irreversible de datos
