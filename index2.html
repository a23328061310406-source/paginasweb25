Tipos de Malware

Conceptos

Esta tabla contiene los principales tipos de malware, su función y el impacto que pueden causar.

  • Definición breve de cada tipo de malware
  • Cómo se comporta / vector de propagación
  • Consecuencias e impacto en sistemas y privacidad
Tipo de Malware Función Impacto
Virus Auto-replicable, infecta archivos y programas. Daño de datos y posible alteración del sistema.
Gusanos (Worms) Se propaga a través de redes sin intervención del usuario. Interrupción de la red y agotamiento de recursos.
Caballo de Troya (Troyano) Se disfraza de software legítimo para engañar al usuario. Robo de datos y control remoto del sistema.
Software espía (Spyware) Supervisa la actividad del usuario en secreto. Violación de la privacidad y robo de información.
Adware (Software publicitario) Muestra anuncios no deseados y puede rastrear comportamiento. Molestia para el usuario y degradación del rendimiento.
Rootkit Oculta la presencia de malware y mantiene acceso privilegiado. Acceso no autorizado y amenazas persistentes difíciles de detectar.
Keylogger Registra las pulsaciones de teclado para capturar credenciales. Robo de contraseñas y filtración de datos personales.
Fileless Malware Opera en memoria sin dejar archivos en disco. Ataques sigilosos y difíciles de detectar por antivirus tradicionales.
Cryptojacker Minado no autorizado de criptomonedas usando recursos del equipo. Rendimiento reducido y consumo excesivo de energía.
Botnet (Red de bots) Red de dispositivos infectados controlados por un atacante. Uso para DDoS, envío masivo de spam y ataques coordinados.
Bomba lógica (Logic Bomb) Ejecuta código malicioso al cumplirse una condición (trigger). Destrucción de datos o daño programado al sistema.
Wiper Malware diseñado para destruir o sobrescribir datos. Pérdida irreversible de información y daño operacional.
